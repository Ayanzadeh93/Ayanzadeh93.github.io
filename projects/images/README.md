# Project Images Directory

## FloorPlan2Guide Diagram

Place your `floorVLM_diag.pdf` file in this directory.

### File Location
- **PDF**: `projects/images/floorVLM_diag.pdf`
- **Image (optional)**: `projects/images/floorVLM_diag.png`

### Instructions

1. Copy your `floorVLM_diag.pdf` file to this directory
2. (Optional) If you want faster loading, convert the PDF to PNG:
   - Use a tool like ImageMagick: `convert floorVLM_diag.pdf floorVLM_diag.png`
   - Or export from PDF viewer as PNG image
   - Place the PNG file in this directory

The page will automatically:
- Try to load the PDF first
- Fall back to PNG image if PDF doesn't load
- Show a download link if neither is available

### File Naming
Make sure the file is named exactly: `floorVLM_diag.pdf` (case-sensitive)
